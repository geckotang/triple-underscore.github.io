12.5. 有名文字参照 — Named character references

HTML Living Standard

下の一覧に， HTML がサポートする文字参照名, および それが指す符号位置を挙げる。 これらは、 HTML 構文 節 から参照される。 This table lists the character reference names that are supported by HTML, and the code points to which they refer. It is referenced by the previous sections.

このデータは JSON ファイル形式 としても入手できる。 この日本語ページでは、このファイルを動的に取得して，下の一覧データを生成している（多少時間を要する場合もあるかもしれない）。 また、 fetch API を利用しているので，未対応ブラウザでは表示されない。 This data is also available as a JSON file.

表示されるグリフは、規範的ではない。 挙げられた文字の公式的な定義は、 Unicode 仕様を参照されたし。 The glyphs displayed above are non-normative. Refer to the Unicode specifications for formal definitions of the characters listed above.

その他の詳細（訳注）：

  • 原文と異なり、既定では符号位置順に整列している — その方が見通しが良いので（下のコントロールで整列順を切り替えれる）。
  • すべての文字参照名は ASCII 英数字のみからなる。
  • 文字参照 自体は、 &文字参照名; のように，アンパサンド（ & ）とセミコロン（ ; ）で括って表記する。
  • 文字参照名の文字大小は区別される。 一部の大小のみ異なり（頭文字に限られない）, よく似たグリフで描画されつつ，異なる文字を指すものもあることに注意。
  • 異なる文字参照名が同じ文字を指す場合もある（例： ast と midast ）。
  • 文字参照名のうち， “†” 印が付いているものは、セミコロンを省略して表記できる（例： "&amp" ）。 が、旧来との互換性のための記法なので，推奨されない。
  • 原文の一覧と異なり，このページでは、すべての文字参照名からセミコロンも省略した上で，前項に述べた省略記法の行も省略している。
  • グリフには、描画されない（不可視の）ものもある。 また、対応するフォントがシステムにない文字は、代用グリフ（ "�" や文字コードを象形化したものなど）で示されるであろう。
  • グリフ描画に利用されるフォントは、ブラウザのデフォルト（セリフフォント）に設定されている。
