この付録は SVG 1.1 第２版 仕様 から SVG 2 仕様に加えられた変更点を要約したものである。 前バージョンの SVG 2 Working Draft （ 2015 年 7 月 9 日）からの変更点には、このスタイルにした上で，記号 “○” を文末に付加している。 This appendix summarizes the changes that have been made since the SVG 1.1 Second Edition Recommendation. Changes made since the last SVG 2 Working Draft are highlighted.

SVG2-L.1. 編集上の変更点

可読性の向上のため、スタイルに関するいくつかの変更が加えられた： A number of stylistic changes have been made to the specification to make it more readable. These include the following:

  • 現今の CSS 仕様に向けたスタイルを利用するように変更。 A change towards using styling similar to contemporary CSS specifications.
  • Working Group のメンバに進行が委任され，取り込みについて解決された特色機能を反映する，注釈が仕様に追加された Annotations have been added to the specification to reflect features that the Working Group has resolved on including and for which a member of the group has committed to drive that feature.
  • 基本データ型章にて comma-wsp, wsp を定義し、他のすべての文法は，これらを参照するようにされた。 Define comma-wsp and wsp in the types chapter and reference these from all other grammars that need it.
  • XML 内のみならず HTML 内でも SVG を利用できることを強調するように変更された。 Changes to highlight that SVG can be used in HTML as well as in XML.
  • メタデータの章と文書構造の章が併合された。 Merged the metadata chapter with the document structure chapter.
  • SVGSVGElement に対する getIntersectionList, getEnclosureList における referenceElement 引数は nullable になるように IDL を更新。 Updated IDL for SVGSVGElement marking the referenceElement argument in getIntersectionList and getEnclosureList as nullable.

SVG2-L.2. 重要な変更点

上に挙げられた編集上の変更に加え、以下の重要な追加, 変更, 削除が行われた。 In additional to the editorial changes listed above, the following substantial additions, changes and removals have been made.

SVG2-L.2.1. 文書全体に渡るもの

  • SVG DTD への参照は削除された。 References to the SVG DTD have been removed.
  • IDL は Web IDL 構文に則するものへ，更新された。 IDL has been updated to use Web IDL syntax.
  • すべてのリスト型インタフェースに "length" 属性と index 付きプロパティ アクセスが追加された。 Added "length" attribute and indexed property access to all list interfaces.
  • <angle> 型と参照 CSS3-values の定義は削除された。 Remove definition of <angle> type and reference CSS3-values.
  • css3-values 仕様を基準にする準備として、すべての <coordinate> 型は <length> に変更された。 Change all coordinate types to <length> in preparation for referencing css3-values..
  • すべての要素の内容モデルに script 要素が追加された。 Add the ‘script’ element to the content model of all elements.
  • css3-values 仕様を基準にする準備として、すべての座標型は <length> に変更された。 Change all coordinate types to <length> in preparation for referencing css3-values..
  • すべての要素の内容モデルに script 要素が追加された。 Add the ‘script’ element to the content model of all elements.
  • リストオブジェクト上の initialize, appendItem, replaceItem, insertItemBefore メソッドを、挿入されつつあるリストアイテムが，別のリスト内にすでにあるときは、複製するように，変更された。 Changed the initialize, appendItem, replaceItem and insertItemBefore methods on list objects to make a copy of any list item being inserted that is already in another list.
  • image, script 要素に、 CORS-enabled fetch アルゴリズムと併用するための crossorigin 属性が加えられた。 The ‘image’ and ‘script’ elements have gained a crossorigin attribute, for use with the CORS-enabled fetch algorithm.

SVG2-L.2.2. （章）概念

  • この章は削除された。 Removed this chapter.

SVG2-L.2.3. （章）描画モデル

  • 描画モデルは， Compositing and Blending 仕様 を参照するように更新された。 Update rendering model to refer to Compositing and Blending Specification.
  • z-index はこの章に移動された。 例を残して参考テキストは削除された。 Moved z-index to this chapter. Removed non-normative text, keeping examples.
  • 課題 2 を解決 — knock-out への参照は削除された。 Resolve issue 2 - removed reference to knock-out.
  • 課題 3 を解決 — 要素とグループがどう描画されるかについての記述は書き直された。 Resolve issue 3 - rewrote description of how elements and groups are rendered.
  • 課題 4 を解決 — 各ペイントサーバ型を明示的に示さないように Resolve issue 4 - Don't mention each paint server type explicity.
  • 課題 6 を解決 — 要求される変更はない。 Resolve issue 6 - No changes required.
  • クリッピング, マスキング, 色合成（ Clipping, Masking and Compositing ）章 をこの章に併合された（それに伴い，課題 7 は解決された）。 Merge Clipping, Masking and Compositing chapter into this chapter (resolved issue 7 in process).
  • “親への合成（ Parent Compositing ）” 節は、各 referencing mode が合成される方法を明示的に示すように更新された。 Update 'Parent Compositing' section to explicitly mention how each referencing mode composites.
  • UA スタイルシートにおいては、 SVG ルート, foreignObject 要素に対しては、最早 overflow:hidden でない。 ○ SVG root and foreignObject are no longer overflow:hidden in UA style sheet.
  • 一部の事例では、 overflow:auto に対し，内容が切り取られてスクロールバーが示されるようになる。 ○ overflow:auto may clip and show scroll bars in some cases.
  • svg 要素に対する overflow:scroll では、スクロールバーが示され得る。 ○ overflow:scroll may show scroll bars on svg elements.
  • 初期クリッピングパスの定義は、除去された。 ○ Removed definition of initial clipping path.

SVG2-L.2.4. （章）基本データ型と基本インタフェース

  • すべての SVGMatrix は、 DOMMatrix または DOMMatrixReadOnly に置換された。 All appearance of SVGMatrix were replaced by DOMMatrix or DOMMatrixReadOnly.
  • すべての SVGRect は、 DOMRect または DOMRectReadOnly に置換された。 All appearance of SVGRect were replaced by DOMRect or DOMRectReadOnly.
  • すべての SVGPoint は、 DOMPoint または DOMPointReadOnly に置換された。 All appearance of SVGPoint were replaced by DOMPoint or DOMPointReadOnly.
  • SVGStylable と SVGLangSpace インタフェースは削除され，それらのすべてのメンバは SVGElement へ移動された。 Removed the SVGStylable and SVGLangSpace interfaces and moved all of their members on to SVGElement.
  • SVGLocatable, SVGTransformable は、 SVGElement を直接的に継承する SVGGraphicsElement へ移行された。 Turned SVGLocatable and SVGTransformable into SVGGraphicsElement, which directly inherits from SVGElement.
  • SVGGraphicsElement インタフェースが追加された。 Added an SVGGraphicsElement interface.
  • isPointInFill, isPointInStroke メソッドを伴う SVGGeometryElement インタフェースが追加された。 Added an SVGGeometryElement interface with isPointInFill and isPointInStroke methods.
  • SVGExternalResourcesRequired インタフェースは削除された。 Removed the SVGExternalResourcesRequired interface.
  • tight ストローク 包含ボックスを取得するための getStrokeBBox が追加された。 Added getStrokeBBox to get the tight stroke bounding box.
  • SVGElement 上の viewportElement, ownerSVGElement は、 nullable （ null 値をとり得る）にされた。 Make viewportElement and ownerSVGElement on SVGElement nullable.
  • SVGElement, SVGColor, SVGICCColor インタフェースから getPresentationAttribute 演算が削除された。 Removed the getPresentationAttribute operation on SVGElement and the SVGColor and SVGICCColor interfaces.
  • SVGElement に， focus, blur 演算と tabIndex 属性が追加された。 Added focus and blur operations and tabIndex attribute to SVGElement.
  • Document に， activeElement 属性が追加された。 Added activeElement attribute to Document.
  • SVGElement は HTML の GlobalEventHandlers インタフェースを実装するようにされた。 Made SVGElement implement the GlobalEventHandlers interface from HTML.
  • SVGGraphicsElement から getStrokeBBox が削除された。 getBBox は、返される包含ボックスに要素のどの部分が含まれるかを制御する，ディクショナリ引数も伴うように拡張された。 Removed getStrokeBBox from SVGGraphicsElement and extended getBBox with a dictionary argument that controls which parts of the element are included in the returned bounding box.
  • <length>, <angle>, <number>, <integer> においては、頭部／尾部の空白類も許容されることに。 Allow leading and trailing whitespace in <length>, <angle>, <number> and <integer>.
  • CSS ／ HTML に揃えるため、 form feed（ U+000C ）も空白類に含まれるようにされた。 Make whitespace include form feed (U+000C) to align with CSS and HTML.
  • 基本データ型のうち、すでに CSS Values and Units に定義されているものは削除された。 Remove basic data types already defined by CSS Values and Units.
  • 色キーワードのリストは削除された。 そのリストは、規範として参照される CSS Colors 3 勧告の一部にされた。 Remove list of color keywords. The list is part of CSS Colors 3 which is referenced normatively and a REC.
  • 種々のリスト型の属性において，値の末尾における単独の comma-wsp も許容されるようになった。 Allow a single trailing comma-wsp in various list-typed attributes.
  • SVGElement インタフェースから xmlbase, xmllang, xmlspace 属性が削除された。 Removed the xmlbase, xmllang and xmlspace attributes from the SVGElement interface.
  • SVGViewSpec インタフェースは削除された。 Removed the SVGViewSpec interface.
  • SVGElement から style IDL 属性が削除された — それは CSSOM にて定義されているので。 Removed the style IDL attribute from SVGElement, since CSSOM defines that for us.
  • SVGGraphicsElement 上の getTransformToElement メソッドは、除去された。 ○ Removed the getTransformToElement method on SVGGraphicsElement.
  • 最も外側の svg 要素 上であっても， getCTM は機能するべきと定義され、それが含むべき座標系変換がリストされた。 ○ Defined that getCTM on an outermost svg element should work, and listed which transforms it should include.
  • 属性への反映について、 SVG DOM オブジェクトが “生きているかどうか（ liveness ）” に及ぼす効果も含め，より深く定義された。 ○ Defined attribute reflection in more depth, including the effect it has on SVG DOM object liveness.

SVG2-L.2.5. （章）文書構造

  • SVGSVGElement インタフェースの suspendRedraw, unsuspendRedraw, unsuspendRedrawAll メソッドは、廃止予定にされた。 Deprecated the suspendRedraw, unsuspendRedraw and unsuspendRedrawAll methods on the SVGSVGElement interface.
  • externalResourcesRequired 属性は、削除された。 Removed the ‘externalResourcesRequired’ attribute.
  • image 要素の width, height 属性に auto を利用する機能が追加された。 Added the ability to use 'auto' for the ‘width’ and ‘height’ attributes on ‘image’.
  • use 要素からは、文書全体は参照できないとする制約は除かれた。 Lifted the restriction on ‘use’ not being able to reference an entire document.
  • desc, title 要素に， lang 属性が追加された。 Added ‘lang’ attribute on ‘desc’ and ‘title’ elements.
  • WAI-ARIA 属性の節が追加された。 Added section on WAI-ARIA attributes.
  • 最も外側の svg 要素 上の CSS transforms は currentScale や currentTranslate に影響しないことが明確化された。 最も外側でない svg 要素上の返り値を定義。 Clarified that CSS transforms on an outermost svg element do not affect currentScale or currentTranslate. Defined the returned values on ‘svg’ elements that are not outermost svg element.
  • rootElement 属性は、廃止予定にされた。 Deprecated the rootElement attribute.
  • HTML body 上のイベントハンドラ内容属性と IDL 属性と同様に、ルート svg 要素上に、 Window に向けて発火されるイベントのための，イベントハンドラ内容属性と IDL 属性が定義された。 Defined event handler content attributes and IDL attributes on the root ‘svg’ element to be for events fired at the Window, like HTML's event handler content attributes and IDL attribute on body.
  • image 要素は、 埋め込み内容 の章に移動された。 Moved the ‘image’ element to the Embedded Content chapter.
  • SVGElementInstance, SVGElementInstanceList インタフェース, および それらに対応する SVGUseElement インタフェース上の属性は削除された。 Removed the SVGElementInstance and SVGElementInstanceList interfaces, and the corresponding attributes on the SVGUseElement interface.
  • use 要素におけるイベントフローは， Shadow DOM 仕様に従うように変更された。 Changed the ‘use’ element event flow to follow the Shadow DOM spec.
  • switch 要素が script 要素の処理に影響しないことが明確化された。 Clarified that the ‘switch’ element does not affect processing of ‘script’ elements.
  • svg 要素上の width, height 属性は呈示属性にされ、それらの初期値は auto に変更された。 Made the ‘width’ and ‘height’ attributes on the ‘svg’ element presentation attributes, and changed the initial value to auto.
  • svg 要素から baseProfile, version 属性が削除された。 Removed the baseProfile and version attributes from the ‘svg’ element.
  • SVGSVGElement インタフェース上の forceRedraw メソッドは廃止予定になり、呼び出されても効果を持たないものと定義された。 Deprecated the forceRedraw method on the SVGSVGElement interface and defined it to have no effect when called.
  • SVGSVGElement インタフェース上の deselectAll メソッドは Selection API の用語を通して定義されると同時に，廃止予定にされた。 Defined the deselectAll method on the SVGSVGElement interface in terms of the Selection API and simultaneously deprecated it.
  • switch 要素は style 要素の処理に影響しないことが明確化された。 Clarified that the ‘switch’ element does not affect the processing of ‘style’ elements.
  • SVGSVGElement インタフェース上の getIntersectionList, getEnclosureList メソッドは、 shadow 木 要素を返さないことが明確化された。 Clarified that the getIntersectionList and getEnclosureList methods on the SVGSVGElement interface do not return shadow tree elements.
  • requiredFeatures 属性は、削除された。 Removed the requiredFeatures attribute.
  • SVGSVGElement 上の currentView, useCurrentView プロパティは、削除された。 Removed the currentView and useCurrentView properties on SVGSVGElement.
  • SVG 名前空間 内の未知の要素に対する取り扱うための， SVGUnknownElement が追加された。 ○ Added SVGUnknownElement for handling unknown elements in the svg namespace.
  • どの名前空間にも属さない lang 属性が追加された。 ○ Added the ‘lang’ attribute in no namespace.
  • SVGSVGElement インタフェースの viewport 属性は、除去された。 ○ Removed the viewport attribute from the SVGSVGElement interface.
  • xml:base 属性は、除去された。 ○ Removed the xml:base attribute.
  • discard 要素は、分離されたモジュール SVG Animations に移行された。 ○ Moved the ‘discard’ element to the split-out SVG Animations module.

SVG2-L.2.6. （章）スタイル付け

  • contentStyleType 属性は、削除された。 Removed the ‘contentStyleType’ attribute.
  • SVGStyleElement は、今や LinkStyle を実装する。 SVGStyleElement now implements LinkStyle.
  • UA スタイルシートは、内側の svg 要素と foreignObject 要素は，最早 overflow:hidden に設定されないとするように更新された。 ○ Updated user agent style sheet - inner svg elements and foreignObject elements are no longer set as overflow:hidden.
  • UA スタイルシートにおいて overflow:hidden とされる要素のリストに， hatch 要素が追加された。 ○ Added hatch to list of elements with overflow:hidden in UA style sheet.
  • ルート svg 要素, および foreignObject 要素の子である svg 要素を除く，すべての要素に対し、 transform-origin に対する既定の値は， 0 0 にされた。 ○ Set the default value of transform-origin to '0 0' for all elements except root svg elements and svg elements that are the child of a foreignObject element.
  • この章は、より簡潔にすべく，大きく書き直された。 ○ Substantially rewrote the chapter to be more concise.
  • HTML link 要素は 外部スタイルシートを読み込まなければならないものと定義された。 ○ Defined that HTML ‘link’ elements must load external style sheets.
  • プロパティの正確な集合が指定された： 呈示属性を持つもの, （呈示属性の名前とプロパティ名が合致しないときの）名前, どの要素がそれらを許容するかについて。 ○ Specified the exact set of properties that have presentation attributes, their names (when the presentation attribute name does not match the property name), and on which elements they are allowed.
  • 適合 SVG 2 UA がサポートしなければならない，他の仕様にて定義されるプロパティのリストが追加された。 ○ Added a list of properties defined in other specifications that conforming SVG 2 user agents must support.
  • 廃止予定にされた xml:space 属性から white-space プロパティへの対応付けを与える UA スタイルシート規則が追加された。 ○ Added UA style sheet rule to map the deprecated ‘xml:space’ attribute to the ‘white-space’ property.
  • text 要素上では、 @font-face 規則と ::first-letter, ::first-line 疑似要素のサポートが要求されるようになった。 ○ Required that @font-face rules and ::first-letter & ::first-line pseudo-elements on ‘text’ elements must be supported.
  • インライン SVG 内容が伴われる HTML 文書における どの SVG ／ HTML スタイルシートも，文書内のすべての内容に適用することが要求されるようになった。 ○ Required that all SVG and HTML style sheets in an HTML document with inline SVG content must apply to all content in the document.

SVG2-L.2.7. （章）幾何的プロパティ

  • x, y プロパティが適用される要素の集合から， pattern, filter が削除された。 Removed ‘pattern’ and ‘filter’ from the set of elements that the ‘x’ and ‘y’ properties apply to.
  • width, height プロパティに対する値 auto は、それらのプロパティが適用される SVG 要素に対しては， 0 に算出されるものと定義された。 ○ Defined that ‘width’ and ‘height’ property values of auto are computed to 0 for SVG elements that the properties apply to.

SVG2-L.2.8. （章）座標系, 変換, 単位

  • SVGMatrix の skewX, skewY メソッドは、不正な値に対し例外を投出するようにされた。 Make SVGMatrix.skew{X,Y} throw an exception on bad values.
  • SVG Tiny 1.2 由来の包含ボックスについてのより良い言い回しに加え，要素の包含ボックスを算出するアルゴリズムが追加された。 Added improved wording on bounding box from SVG Tiny 1.2, and an algorithm which can compute a bounding box for an element.
  • 位置を持たない要素に対する 包含ボックスは ( 0, 0 ) に位置するものと定義された。 Define bounding box for element with no position to be at (0,0).
  • preserveAspectRatio から defer キーワードが削除された。 Removed the defer keyword from ‘preserveAspectRatio’.
  • “現在のビューポート” のための定義が追加された。 SVG ビューポートは、各種 SVG 要素により作成された任意のビューポートを意味するように変更された。 ○ Add definition for 'current viewport'. Change SVG viewport to mean all viewports created by SVG elements.

SVG2-L.2.9. （章）パス

  • 「方角（ bearing ）」を指示する 新たなパス命令として、 B と b が追加された。 Added new B and b "bearing" path commands.
  • Z ／ z 命令は、前の命令にて欠落しているパス座標データを，その部分パス内の最初の座標の座標成分で補完できるようにされた。 Allow Z or z to fill in missing path coordinate data in the previous command with the coordinate of the initial point in the subpath.
  • データ集合を伴わない（空または妥当な命令がない）ような path, polygon, polyline 要素は、描画しないものと定義された。 Define ‘path’, ‘polygon’ and ‘polyline’ elements with no data set (empty or zero valid commands) to not render.
  • SVGPathSeg*, SVGAnimatedPathData インタフェース, および SVGPathElement 上の関連のメソッドは削除された。 Removed the SVGPathSeg* and SVGAnimatedPathData interfaces and the related methods on SVGPathElement.

SVG2-L.2.10. （章）テキスト

  • text, textPath 要素の要約ボックスに欠けていたいくつかの属性が追加された。 Added a number of missing attributes to the element summary boxes of the ‘text’ and ‘textPath’ elements.
  • text 要素に inline-size 呈示属性が追加された。 text-overflow 処理についての節が追加された。 Added ‘inline-size’ presentation attribute to ‘text’, and a section about ‘text-overflow’ processing.
  • white-space プロパティが追加された。 xml:space 属性は、廃止予定にされた。 Added ‘white-space’ property and deprecated ‘xml:space’ attribute.
  • kerning プロパティは、削除された。 Removed the ‘kerning’ property.
  • textPath 要素に d 属性が追加された。 Added ‘d’ attribute to ‘textPath’.
  • textPath には、基本図形を参照する機能が追加された。 Added ability to reference basic shapes to ‘textPath’.
  • tref 要素は、削除された。 Removed the 'tref' element.
  • altGlyph, altGlyphDef, altGlyphItem, glyphRef 要素は、削除された。 Removed the 'altGlyph', 'altGlyphDef', 'altGlyphItem' and 'glyphRef' elements.
  • SVGTextContentElement インタフェース上の selectSubString メソッドは、 Selection API の用語を通して定義されると同時に，廃止予定にされた。 Defined the selectSubString method on the SVGTextContentElement interface in terms of the Selection API and simultaneously deprecated it.
  • getComputedTextLength() は、 dx, dy 値を含めないように変更された。 ○ Changed getComputedTextLength() to not include ‘dx’ and ‘dy’ values.
  • width, height プロパティに関する論は、幾何的プロパティ章へ移行された。 ○ Moved the discussion of ‘width’ and ‘height’ properties to the Geometry Properties chapter.

SVG2-L.2.11. （章）埋め込み内容

  • この章を追加： HTML のものと ほぼ同等にふるまう，新たな要素： video, audio, iframe, canvas, source, track が追加された。 Added this chapter, which includes new ‘video’, ‘audio’, ‘iframe’, ‘canvas’, ‘source’ and ‘track’ elements that behave almost identically to the equivalent HTML elements.
  • HTML の埋め込み内容 要素に対応する， SVG による複製は、除去された。 代わって、それらの要素は， SVG コンテナ要素の中であっても， HTML 名前空間の下で利用できるようにされた。 ○ Removed the SVG copies of HTML's embedded content elements and instead allowed those elements to be used in the HTML namespace within SVG container elements.
  • image 要素から参照されている SVG 文書上の clip, overflow は無視するとする要件は、除去された。 ○ Removed the requirement to ignore ‘clip’ and ‘overflow’ on an SVG document when referenced by an ‘image’ element.

SVG2-L.2.12. （章）塗り

  • ストローク プロパティについて，より詳細な記述が追加された。 ストロークが持たなければならない正確な形状を定義するアルゴリズムが追加された。 Added more detail to the descriptions of the stroke properties and added algorithms defining the exact shape a stroke must have.
  • paint-order プロパティが追加された。 Added the ‘paint-order’ property.
  • color-interpolation-filters プロパティは Filter Effects 仕様へ移動された。 Moved the color-interpolation-filters property to the Filter Effects specification.
  • buffered-rendering プロパティが追加された。 Added the ‘buffered-rendering’ property.
  • 塗りの指定に値 context-fill, context-stroke が追加された。 Added the context-fill and context-stroke paint values.
  • <paint> 値とマーカプロパティは、 ID を利用しない参照をサポートするため， child キーワードと <child-selector> 値をとれるようにされた。 Allowed <paint> values and marker properties to take the child keyword and <child-selector> values to support ID-less referencing.
  • 拡縮されないストロークをサポートするため、 vector-effect プロパティが追加された。 Added ‘vector-effect’ property to support non-scaling stroke.
  • stroke-linejoin がとり得る値に arcs が追加された。 Added arcs value on ‘stroke-linejoin’.
  • marker 要素の orient 属性に auto-start-reverse が追加された。 Added the 'auto-start-reverse' attribute to the ‘orient’ attribute on ‘marker’.
  • SVGPaint インタフェースは、削除された。 Removed the SVGPaint interface.
  • fill, stroke の <paint> の定義を，複数の塗りもとり得るように拡張された。 Extended the definitions of ‘fill’ and ‘stroke’ <paint> so that they can take multiple paints.
  • z-index プロパティが追加された。 Added the ‘z-index’ property.
  • 新たなマーカ／ストローク関連の特色機能の一部は、 SVG Markers, SVG Strokes 仕様に分離された。 Split out some new marker and stroke related features into the SVG Markers and SVG Strokes specifications.
  • <child-selector> は、塗り値から除去された。 ○ Removed <child-selector> paint values.

SVG2-L.2.13. （章）色

  • 画像が色管理の対象になることについての要件が追加された。 Added requirements for images to be color managed.
  • SVG Color 仕様から色の構文が新たに追加された： RGBA 色, HSL 色, LAB 色, ICC 名による色, デバイス色。 Added new color syntaxes from the SVG Color specification: RGBA colors, HSL colors, LAB colors, ICC named colors and device colors.
  • color-profile プロパティは、削除された。 Removed the ‘color-profile’ property.
  • color-profile 要素は、削除された。 Removed the ‘color-profile’ element.

SVG2-L.2.14. （章）ペイントサーバ

  • SVG Tiny 1.2 からの移入により， solidcolor 要素とその２つのプロパティ solid-color, solid-opacity が追加された。 （ 'solidColor' は 'solidcolor' に改称された。） Added the ‘solidcolor’ element and its two properties ‘solid-color’ and ‘solid-opacity’, ported over from SVG Tiny 1.2. (Renamed 'solidColor' to 'solidcolor'.)
  • radialGradient 要素に，焦点円の半径を指定可能にする fr 属性が追加された。 Added an ‘fr’ attribute to the ‘radialGradient’ element, which allows specifying the radius of the focal circle.
  • mesh, meshrow, meshpatch 要素を定義する，メッシュグラデーションの節が追加された。 （ 'meshGradient' は 'mesh' に, 'meshRow' は 'meshrow' に, 'meshPatch' は 'meshpatch' に改称された。） Added a section for mesh gradients, defining the ‘mesh’, ‘meshrow’ and ‘meshpatch’ elements. (Renamed 'meshGradient' to 'mesh', 'meshRow' to 'meshrow', and 'meshPatch' to 'meshpatch'.)
  • hatch, hatchpath 要素を定義するための hatch 節 が追加された。 （ 'hatchPath' は 'hatchpath' に改称された。） Added a section for hatches, defining the ‘hatch’ and ‘hatchPath’ elements. (Renamed 'hatchPath' to 'hatchpath'.)
  • mesh 要素上の gradientTransform, および hatch 要素上の hatchTransform は， transform に改称された。 Renamed ‘gradientTransform’ on the ‘mesh’ element and ‘hatchTransform’ on the ‘hatch’ element to ‘transform’.

SVG2-L.2.15. （章）クリッピング, マスキング, 色合成

  • SVG は今や CSS Masking [CSS-MASKING] 仕様を参照する。 SVG now references CSS Masking [CSS-MASKING] specification.

    • クリッピング／マスキングのプロパティや要素の定義は削除された。 Removed definition of clipping/masking properties and elements.
    • クリッピング／マスキングについての概説は残され、 CSS Masking への参照が追加された。 Keep short introduction to clipping/masking and add reference to CSS Masking
  • Compositing and Blending （色の合成と混色）仕様への参照が追加された。 単純アルファ合成のサポートが義務化された。 Added reference to Compositing and Blending spec. Simple alpha compositing it mandatory.
  • html にインラインの，最も外側の svg 要素上では、 overflow プロパティが尊重されるべきとされた。 The ‘overflow’ property should be respected on the outermost svg elements inline in html.
  • クリッピング, マスキング, 色合成 （ Clipping, Masking and Compositing ）章は、描画モデル（ Rendering Model ）章に併合された。 Merge Clipping, Masking and Compositing chapter into Rendering Model chapter.

SVG2-L.2.16. （章）フィルタ効果

SVG2-L.2.17. （章）対話性

  • tabindex 属性が追加された。 Added the ‘tabindex’ attribute.
  • cursor 要素上の href 属性の定義は、 CSS 式の関数形でない URL をとるように修正された。 Corrected the definition of the ‘href’ attribute on the ‘cursor’ element to take an URL that is not in a CSS-like functional form.
  • pointer-events に，キーワード bounding-box が追加された。 Added the bounding-box keyword to ‘pointer-events’.
  • SVGLoad, SVGAbort, SVGError, SVGUnload は，それぞれ load, abort, error, unload に置換された。 Replaced SVGLoad, SVGAbort, SVGError and SVGUnload with load, abort, error and unload respectively.
  • load イベントは、 構造的に外部の要素 と, 最も外側の svg 要素 のみに，発火されなければならないことが要件とされた。 Required that only structurally external elements and the outermost svg element must fire load events.
  • SVGResize, SVGScroll は、それぞれ resize, scroll に置換された。 Replaced SVGResize and SVGScroll with resize and scroll respectively.
  • DOMActivate は、削除された。 Removed DOMActivate.

  • mutation イベントは、削除された。 Removed mutation events.

SVG2-L.2.18. （章）リンク

  • view 要素の内容モデルに アニメーション要素 が追加された。 Added animation elements to the content model of ‘view’.
  • Media Fragments 仕様にて定義される素片識別子を伴う SVG リソースへのリンクについての可能性が追加された。 Added the possibility to link to SVG resources with fragment identifiers defined by the Media Fragments specification.
  • xlink:type, xlink:role, xlink:arcrole, xlink:show, xlink:actuate 属性は、削除された。 Removed the ‘xlink:type’, ‘xlink:role’, ‘xlink:arcrole’, ‘xlink:show’ and ‘xlink:actuate’ attributes.
  • xlink:href 属性は、名前空間なしの href の利用の支持を受けて，廃止予定にされた。 Deprecated the ‘xlink:href’ attribute in favor of using ‘href’ without a namespace.
  • xlink:title 属性は、 title 子要素の利用の支持を受けて，廃止予定にされた。 Deprecated the ‘xlink:title’ attribute in favor of using child ‘title’ elements.
  • SVG ビュー素片では、スペース文字も許容されるようになった。 ○ Allowed spaces in SVG view fragments.
  • "pixel:", "percent:" による，空間的な部位を指示するメディア素片を解釈する方法が定義された。 ○ Defined how "pixel:" and "percent:" spatial media fragments are interpreted.
  • view 要素へのリンクは、最も近い先祖の svg 要素を示すためにビューボックスの座標系を暗黙的に変換することは、ないようにされた。 ○ Made linking to ‘view’ elements not cause an implicit view box transformation to show the nearest ancestor ‘svg’ element.
  • SVG ビュー素片パラメタは、対応する属性を初期値にリセットしないことが明確化された。 それまで未指定だった。 ○ Clarified that unspecified SVG view fragment parameters do not cause the corresponding attributes to be reset to their initial values.
  • view 要素の viewTarget 属性と対応する SVG ビュー素片パラメタは、除去された。 ○ Removed the ‘viewTarget’ attribute on the ‘view’ element and the corresponding SVG view fragment parameter.
